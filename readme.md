# Easy Learn Digital Image Processing 

# 项目简介
本项目旨在通过python教会大家掌握数字图像处理的技术，能够帮助大家快速了解图像切割，灰度图像处理，彩色图像处理，对比度处理的原理，方便0基础入门数字图像处理。
